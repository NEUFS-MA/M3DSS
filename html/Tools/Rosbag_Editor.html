Rosbag Editor

1. The overview of Rosbag Editor

Rosbag Editor is a very powerful editor for rosbag that can delete topics, cut length, rename topics, and other powerful functions from rosbag. Below, we will introduce its usage process.

2. Install

The installation process is very simple:

(1)mkdirs catkin_ws/src
(2)cd catkin_ws/src
(3)git clone https://github.com/facontidavide/rosbag_editor
(4)cd .. && catkin build

After the compilation is completed, the Rosbag Editor is successfully installed, and then enter the following command to run it successfully
rosrun rosbag_editor rosbag_editor

3.Application Example
